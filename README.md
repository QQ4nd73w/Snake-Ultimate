# Snake

Простая реализация игры «Змейка» на HTML/CSS/JavaScript

Файлы:
- [index.html](index.html) — структура страницы и холст.
- [style.css](style.css) — базовые стили.
- [script.js](script.js) — логика игры.

Запуск: откройте `index.html` в браузере через Live Server.

Рекорды:
- Лучший результат сохраняется в `localStorage` и показывается как `Best` на странице.

Идеи для доработок:
- Добавить уровни сложности (ускорение со временем).
- Кнопки управления на экране для мобильных устройств.
- Эффекты и анимация при съедании еды.
- Таблица рекордов и загрузка/сохранение через backend.
